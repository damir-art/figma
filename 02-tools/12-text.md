# Text (текст) `T`
Руководство по Figma: https://figma.su

Текст это шейп внутри которого расположен текст.

Создание текстового слоя:
- Жмем `T`, жмем по рабочей области Figma
- Жмем `T`, левой кнопкой мыши обозначаем область

## Свойства текстового слоя
- Название шрифта
- Стиль шрифта
- Размер шрифта
- Высота шрифта, обычно ставят 130%
- Межбуквенный интервал (трекинг)
- Расстояние между абзацами, разделёнными через `Enter`
- Режимы текстового слоя: Auto Width, Auto Height, Fixed Size
- Выравнивание текста: по горизонтали/вертикали
- Расширенные настройки

## Режимы текстового слоя
- Auto Width - ширина текстового блока подстраивается под контент
- Auto Height - высота текстового блока подстраивается под контент
- Fixed Size - шириная текстового блока фиксирована (используется в компонентах)

## Горячие клавиши
    Text Align Left    - Cmd + Alt + L
    Text Align Center  - Cmd + Alt + T
    Text Align Right   - Cmd + Alt + R
    Text Align Justify - Cmd + Alt + J

## Разное
* Размеры текстовой области (рамок) можно регулировать.
* Можно сразу задать размеры текстовой области, а затем вставить туда текст.
* **Плагин:** Better Font Picker - показывает как выглядит шрифт
* Тексту можно добавить цвет, обводку, тень.
* По-умолчанию Figma использует Google-шрифты, даже если они у вас не установлены.
* Выдели текстовый слой: `Ctrl + B` - жирный, `Ctrl + I` - курсив

**Шрифты:** display, handwriting, monospace - обычно для заголовков.
